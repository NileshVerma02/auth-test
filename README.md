# auth-test
ROR demo app for simple authentication and authorization

## Development Setup

### requirements
Ruby version 2.7.1
Rails version 6.0.3.4
Node version 12.18.1

### Environment setup steps
Clone the repo from github
Install ruby
Create Gemset named as ror_test
Bundle install
Install Node
Run webpacker command
